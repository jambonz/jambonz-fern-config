### Use the API

To add a plant, use the `POST /plant` endpoint. Be sure to add the required `name` parameter: 

<ParamField path="name" type="string" required={true}>
The name of the plant you would like to add. 
</ParamField>

### Use the SDK

To add a plant, use the `createPlant` method. Be sure to add the required `name` parameter. 

<ParamField path="name" type="string" required={true}>
The name of the plant you would like to add. 
</ParamField>

**Example request and response**

```js 
plantStore.createPlant({
  name: 'Monstera Deliciosa',
  price: 24.99,
  stock: 10,
  category: 'Indoor Plants',
  description: 'A popular tropical plant with large, split leaves.'
})
  .then(plant => {
    console.log('Plant created:', plant);
  })
  .catch(error => {
    console.error('Error creating plant:', error);
  });
```

You can expect a response like: 

```json Response
{
  "id": "plant-003",
  "name": "Monstera Deliciosa",
  "price": 24.99,
  "stock": 10,
  "category": "Indoor Plants",
  "description": "A popular tropical plant with large, split leaves.",
  "createdAt": "2024-10-22T12:34:56Z"
}
```
