---
title: Say
subtitle: Generate text-to-speech audio.
---

```json
{
  "verb": "say",
  "text": "hi there!",
  "synthesizer" : {
    "vendor": "google",
    "language": "en-US"
  }
}
```

## Parameters

<ParamField path="earlyMedia" type="boolean" required={false}>
  If true and the call has not yet been answered, play the audio without answering the call.  
  Default: `false`.
</ParamField>

<ParamField path="loop" type="number" required={false}>
  The number of times a text is to be repeated.  
  `0` means repeat forever.  
  Default: `1`.
</ParamField>

<ParamField path="synthesizer.gender" type="string" required={false}>
  (Google only) `MALE`, `FEMALE`, or `NEUTRAL`.
</ParamField>

<ParamField path="synthesizer.language" type="string" required={false}>
  Language code to use.
</ParamField>

<ParamField path="synthesizer.vendor" type="string" required={false}>
  Speech vendor to use (see list below, along with any others added via the  
  [custom speech API](/guides/speech-providers/custom-speech-api)).
</ParamField>

<ParamField path="synthesizer.voice" type="string" required={false}>
  Voice to use.  
  Note that the voice list differs depending on whether you are using AWS or Google.  
  Defaults to application setting, if provided.
</ParamField>

<ParamField path="text" type="string" required={true}>
  Text to speak; may contain SSML tags.
</ParamField>
