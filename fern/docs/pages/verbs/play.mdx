---
title: Play
subtitle: Stream recorded audio to a call.
---


```json
{
  "verb": "play",
  "url": "https://example.com/example.mp3",
  "timeoutSecs": 10,
  "seekOffset": 8000,
  "actionHook": "/play/action"
}
```

### Parameters

<ParamField path="url" type="string | array" required={true}>
  A single URL or an array of URLs (will play in sequence) to a WAV or MP3 file.
</ParamField>

<ParamField path="actionHook" type="string" required={false}>
  Webhook that is called when the play verb completes.
</ParamField>

<ParamField path="earlyMedia" type="boolean" required={false} default="false">
  If true and the call has not yet been answered, play the audio without answering the call.  
</ParamField>

<ParamField path="loop" type="number" required={false} default="1">
  Number of times to play the URL(s).  
</ParamField>

<ParamField path="seekOffset" type="number" required={false}>
  Number of samples to seek into the URL.
</ParamField>

<ParamField path="timeoutSecs" type="number" required={false}>
  Number of seconds to play the URL.
</ParamField>

### actionHook properties

The actionHook that is invoked when the `play` command completes will include the following properties:

| property name  | description | 
| ------------- |-------------|
| playback_seconds | the number of seconds played |
| playback_milliseconds | the number of milliseconds played |
| playback_last_offset_pos | the last offset position in samples |
