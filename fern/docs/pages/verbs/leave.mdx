---
title: Leave
subtitle: Transfers a call out of a queue
---

The call will return to the flow of execution following the [enqueue](/verbs/verbs/enqueue) 
verb that parked the call, or the document returned by that verbs *actionHook* property, 
if provided.

There are no parameters for the `leave` verb.

```json
{
  "verb": "leave"
}
```
