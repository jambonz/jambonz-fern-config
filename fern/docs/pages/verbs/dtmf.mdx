---
title: Dtmf
subtitle: Sends outgoing DTMF signals on a call; useful for navigating IVRs or voicemail systems.
---

```json
{
  "verb": "dtmf",
  "dtmf": "0276",
  "duration": 250
}
```

## Parameters

<ParamField path="dtmf" type="string" required={true}>
  A string containing a sequence of DTMF digits (0-9, *, #).
</ParamField>

<ParamField path="duration" type="number" required={false}>
  The length of each digit, in milliseconds. Defaults to 500.
</ParamField>
