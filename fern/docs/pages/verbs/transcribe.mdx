---
title: Transcribe
subtitle: Send real time transcriptions of speech continuously.
hide-toc: true
---

<Note>
The transcribe verb may appear on its own or as a nested verb within a dial or listen verb.
</Note>

```json
{
  "verb": "transcribe",
  "transcriptionHook": "http://example.com/transcribe",
  "recognizer": {
    "vendor": "Google",
    "language" : "en-US",
    "interim": true
  }
}
```

You can use the following options in the `transcribe` command:

| option        | description | required  |
| ------------- |-------------| -----|
| recognizer.transcriptionHook | Webhook to receive an HTPP POST when an interim or final transcription is received. | yes |
| [recognizer](/verbs/verbs/recognizer) | Speech recognition options | yes |
