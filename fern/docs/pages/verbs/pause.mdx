---
title: Pause
subtitle: Waits silently for a specified number of seconds.  
hide-toc: true
---

```json
{
  "verb": "pause",
  "length": 3
}
```

You can use the following options in the `pause` action:

| option        | description | required  |
| ------------- |-------------| -----|
| length | number of seconds to wait before continuing the app | yes |
