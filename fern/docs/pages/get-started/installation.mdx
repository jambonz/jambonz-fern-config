This guide will walk you through installing the **Plant Store API SDK** to help you easily integrate with the Plant Store platform.

## Requirements

Before installing the SDK, make sure you have the following:

- **Node.js**: Ensure you have [Node.js](https://nodejs.org/) installed (version 14 or later).
- **Package Manager**: You'll need a package manager such as npm, pnpm, or yarn.

## Install the SDK

<CodeBlocks>
```bash npm
npm install plant-store-api-sdk
```
```bash pnpm
pnpm add plant-store-api-sdk
```
```bash yarn
yarn add plant-store-api-sdk
```
</CodeBlocks>

For more advanced configuration options or troubleshooting, visit our [FAQs](/docs/resources/faqs).

## Update the SDK

It's important to keep the SDK up to date to ensure you're using the latest features and security fixes. Use the following commands to update the Plant Store SDK to the latest version.

<CodeBlocks>
```bash npm
npm update plant-store-api-sdk
```
```bash pnpm
pnpm update plant-store-api-sdk
```
```bash yarn
yarn upgrade plant-store-api-sdk
```
</CodeBlocks>

## Next Steps
After successfully installing or updating the SDK, you're ready to integrate it into your project. For more details on how to configure and use the SDK, check out the [Quickstart Guide](/docs/get-started/quickstart) or the [API Reference](/reference/endpoints).
